Welcome to eTuneBook


With eTuneBook you can

  • Manage Abc tunes in your browser.
  • Build tune sets.
  • Label sets with regard to target (e.g. ceili), environment (e.g. band name) or name (e.g. set-dance name).
  • Specify repeat time or add a comment to set positions.
  • View and print single sets showing the first few bars of each tune.
  • Remember when you last played a tune and how good you were.
  • Label tunes with a color or add a comment.
  • Transpose tunes.
  • Print single tunes.
  • Play tunes on a MIDI Instrument. (currently disabled. will be re-introduced soon...)
  • Store links to other sites holding tune informations.
  • Store links to YouTube videos and view them in eTuneBook directly.
  • Easily identify those tunes you should rehearse. Just sort by last modification date, last play date, rehearse frequency or skill.
  • Search tunes by title, key or type.
  • Search sets by tune type, tune key, tune color, tune skill, set target or set environment.
  • Import Abc tunes from a File on your hard disk.
  • Export Abc tunes to a File on your hard disk.

Please use Chrome when working with eTuneBook.
Other Browsers might work, but they are much slower and might not support all features of eTuneBook.
